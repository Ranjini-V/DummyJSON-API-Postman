# Overview

This repository contains a Postman API testing collection created to practice and demonstrate end-to-end API testing concepts using a public API.
The collection focuses on functional validation, data-driven testing, environment management, and automation-ready practices.

# Key Features Covered

-  CRUD operations testing
-  Status code & response time validation
-  Response body field validation
-  Dynamic variable handling
-  Pre-request & test scripts
-  Schema & field presence validation
-  Data-driven user creation
